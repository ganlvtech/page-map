A mini map for WordPress page.

== Description ==

Add a mini map to your site like Sublime mini map bar.

This WordPress plugin is powered by Lars Jung's pagemap.js library, and published to WordPress plugins repository by Ganlv.

Library hompage: https://larsjung.de/pagemap/
Library GitHub repo: https://github.com/lrsjng/pagemap
Plugin GitHub repo: https://github.com/ganlvtech/page-map

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/page-map/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
